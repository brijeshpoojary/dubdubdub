Styleguide

CSS

The project uses Sass to provide the CSS with some structure.

  • You need to install Ruby (unfortunately) and Sass
  • Watching and compiling can be done by running one single command: sass --watch assets/static/sass/style.scss:assets/static/css/style.css. Doing this outside the vagrant box is a bit faster (practically realtime) and is hence recommended.
  • Don't edit the generated style.css in /assets/static/css
