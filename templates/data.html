List of data sets

KLP Schools

This data set is from the KLP Database and contains a list of all public primary schools KLP partners have worked with and are currently known to be active and functional. It includes the KLP ID, the DISE ID (where known) along with the hierarchy information, address, landmarks, identifiers, associated MP/MLA/Corporator information and geo-codes for most. This data set was last updated in 2013.

Download

KLP Pre-Schools

This data set is from the KLP Database and contains a list of all anganwadis and some private ballads KLP partners have worked with and are currently known to be active and functional. It includes the KLP ID along with the hierarchy information, address, landmarks, identifiers, associated MP/MLA/Corporator information and geo-codes for most. This data set was last updated in 2013.

Download

DISE Schools

These data sets are from NUEPA DISE Database and contains a list of all public, private and aided primary schools that are currently known to be active and functional. It includes the DISE ID along with the hierarchy information, associated MP/MLA/Corporator information and geo-codes for most. This data set was last updated in 2013 and is available for three years.

Download

Other Datasets

KLP has multiple other datasets including data on learning assessments, infrastructure and mid-day meals. These are currently available on request and will be available on this page once we have refactored our database and write clean APIs for them.

KLP APIs

The site is powered by APIs. You can browse the KLP API and the DISE API. Documentation is spotty but we will improve this over the next few months.
